Prisma command:

```bash
npx prisma init

npx prisma generate

```

Husky command:

```bash
npx husky init

npm run prepare

chmod +x .husky/*
# kemudian buat file nya di folder .husky
```

Jest command:

```bash
npm install -D jest jest-environment-jsdom @testing-library/react @testing-library/jest-dom @types/jest ts-node

npm init jest@latest
```
